How this tool works

Step 1: The app queries NewsAPI to pull in a bunch of news headlines from the country that user chooses.

Step 2: It feeds those headlines to a sentiment analysis package, (currently, "VADER"). If the headlines are in a non-English language, I pass those headlines to Google's Cloud Translation API first. The package then conducts a sentiment analysis on the headlines (rating them from -1 to 1, where -1 is very negative, and 1 is very positive (0 is neutral)).

Step 3: Finally, the app calculates a "bad to good headlines" ratio, and using a pre-defined formula, along with the tolerance of bad news provided by the user, recommends whether or not the user should read the news today!


Other notes:

Previously, the model was using meta's llama 2 AI model to analyze, but that got expensive to run. I'm considering re-releasing the app using this model, and if you want to help support that effort, feel free to buy me a coffee :)

The selection of countries that are supported are due to the availability of countries being scraped by the API I am using to grab daily news headlines: NewsAPI!
